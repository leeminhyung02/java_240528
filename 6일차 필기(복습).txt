배열
- 같은 자료형과 같은 의미를 가지는 변수가 여러개 필요한 경우 사용

배열 선언
자료형 [] 배열명 = new 자료형 [크기];
자료형 배열명 [] = new 자료형 [크기];

자료형 [] 배열명 = new 자료형 [] {원소1,원소2};

-지역변수는 초기화가 된다(X)
-배열을 new 연산자를 이용하여 생성하면 생성된 각 배열들은 초기화 된다(O)

배열을 변수처럼 사용하려면 배열명 [번지]
번지는 0번지부터 크기-1번지 까지 사용 가능
실수로 크기번지에 접근하면 ArrayIndexOutOfBoundsException
배열의 길이 : 배열명.length

향상된 for문
배열 또는 컬렉션에서 각 값들 전체를 탐색할 때 사용
1. 전체 탐색(중간에 나갈 수 있다. break를 이용하여)
2. 각 값들을 수정할 수 없다.
